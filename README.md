# 🧾 ReservationSystem

Système de réservation développé en ASP.NET Core basé sur une architecture orientée services.

---

## 🚀 Fonctionnalités

- Authentification des utilisateurs (JWT)
- Création et gestion des réservations
- Gestion des ressources (ex: salles, services)
- Paiement intégré (Stripe)
- API Gateway pour centraliser les requêtes
- Architecture microservices

---

## 🛠️ Technologies utilisées

- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / MySQL
- Docker
- Kubernetes
- Ocelot (API Gateway)
- Postman / Swagger
